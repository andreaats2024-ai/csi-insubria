📤 Modalità di Condivisione in OneDrive

Scegli il tipo di accesso corretto prima di condividere qualsiasi file o cartella.

👤
Persone Specifiche
Solo le persone indicate per nome o email potranno accedere al file. Massimo controllo sulla riservatezza.
✔ Consigliato per dati sensibili
🏢
Tutta l'Organizzazione
Chiunque abbia un account ATS Insubria può visualizzare o modificare il file tramite il link.
⚠ Usare con cautela
🌐
Chiunque abbia il Link
Accesso aperto a chiunque riceva il link, anche senza account Microsoft. Nessun controllo identità.
✖ Non usare per documenti riservati
💡 Buona prassi per ATS Insubria
Impostare sempre una data di scadenza sul link condiviso e scegliere i permessi "Solo visualizzazione" se il destinatario non deve modificare il documento.
1
Cliccare con il tasto destro sul file → Condividi
2
Scegliere la modalità di accesso appropriata
3
Impostare i permessi: Modifica oppure Solo visualizzazione
4
Aggiungere una data di scadenza e una password (facoltativa)
5
Cliccare su Invia o copiare il link
