I file avranno questi stati in Esplora file:

☁

Un'icona di nuvola blu accanto a un file o cartella OneDrive indica che il file è disponibile unicamente online. I file solo online non occupano spazio sul dispositivo. Verrà visualizzata un'icona di nuvola per ogni file solo online in Esplora file, ma il file non verrà scaricato sul dispositivo finché non viene aperto. I file solo online non possono essere aperti quando il dispositivo non è connesso a Internet.

✓

Quando si apre un file solo online, questo viene scaricato sul dispositivo e diventa un file disponibile localmente. È possibile aprire un file disponibile localmente in qualsiasi momento, anche senza accesso a Internet. Se si ha bisogno di più spazio, è possibile riportare il file allo stato solo online: fare clic con il tasto destro sul file e selezionare "Libera spazio".

Con il sensore di archiviazione attivato, questi file torneranno disponibili solo online dopo il periodo di tempo selezionato.

✓

Solo i file contrassegnati come "Mantieni sempre su questo dispositivo" hanno il cerchio verde pieno con la spunta bianca. Questi file sempre disponibili vengono scaricati sul dispositivo e occupano spazio, ma sono sempre accessibili anche quando si lavora senza connessione.
