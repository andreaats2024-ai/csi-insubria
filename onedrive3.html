🔄 Icone di Sincronizzazione OneDrive — Guida Rapida

Icona Stato Significato e Azione Consigliata
☁️ Solo Online Il file esiste nel cloud ma non è scaricato localmente. Aprirlo richiede connessione Internet.
✅ Disponibile Localmente File scaricato e sincronizzato. Accessibile anche offline. Può tornare solo online con "Libera spazio".
💚 Sempre Disponibile File contrassegnato per restare sempre sul dispositivo. Occupa spazio ma è sempre accessibile.
🔄 Sincronizzazione in Corso OneDrive sta caricando o scaricando il file. Attendere il completamento prima di chiudere il PC.
⚠️ Avviso Problema non bloccante. Solitamente spazio quasi esaurito o conflitto di versione. Controllare le notifiche.
❌ Errore di Sincronizzazione Il file non si sincronizza. Verificare connessione Internet o se il file è bloccato da un'altra app.
🔒 File Bloccato Il file è aperto in modalità esclusiva da un altro utente. Attendere che venga chiuso.
🔁 Conflitto di Versione Due versioni dello stesso file in conflitto. OneDrive crea una copia con suffisso del nome utente.
💡 Suggerimento: Se la sincronizzazione rimane bloccata, cliccare con il tasto destro sull'icona OneDrive nella barra delle notifiche → Impostazioni → Chiudi OneDrive, poi riavviare l'app.
